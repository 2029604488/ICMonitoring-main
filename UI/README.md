# ICMonitoring-UI

## 怎么运行这个前端项目？

### 第零步

首先需要安装Node环境，搭建方法:

https://www.runoob.com/nodejs/nodejs-install-setup.html

### 第一步

```shell
npm install
```

### 第二步
```
npm run serve
```

### 第三步

- 访问Local:   http://localhost:8080

## 如果你想部署到服务器

如果你想部署到服务器的话，运行以下命令：

```
npm run build
```

然后你会发现有个dist目录，把他下面的内容放到你的nginx目录中就ok了！
